# Day 15 – Vector Projection

This folder contains Python programs for calculating and visualizing
vector projection.

---

## Files Included

- projection_calc.py – Calculates scalar and vector projection
- projection_plot.py – Visualizes vector projection
- day15_log.md – Daily learning log

---

## Concepts Covered

- Scalar projection
- Vector projection
- Dot product application
- Component resolution

---

## Libraries Used

- math
- matplotlib
